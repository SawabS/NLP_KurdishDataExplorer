# Kurdish Data Explorer — Final Report

This directory contains the submission-ready report and the live-system
screenshots used as evidence.

## Build

From this directory:

```bash
latexmk -lualatex -interaction=nonstopmode -halt-on-error main.tex
```

The final deliverable is:

```text
Kurdish_Data_Explorer_Final_Report.pdf
```

The report is grounded in:

- the instructor's `Final Report Requirements - Tagged.pdf`;
- the implemented code and saved artifact metadata;
- the deployed application at <https://kdx-explorer.fly.dev>;
- the corpus provenance repository at
  <https://github.com/SawabS/Bahdini_Crawler>.

Screenshots were captured from the live deployment on July 24, 2026 at a
1600×1000 browser viewport.
